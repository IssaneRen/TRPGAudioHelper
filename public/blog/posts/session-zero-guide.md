# Session Zero 完全指南

Session Zero 是正式跑团前的准备会议，是一次成功 Campaign 的基石。

## 为什么需要 Session Zero？

很多团因为**期望不匹配**而崩溃：

- 有人想严肃恐怖，有人想搞笑欢乐
- 有人喜欢战斗，有人喜欢角色扮演
- 有人接受角色死亡，有人完全无法接受

Session Zero 就是把这些潜在冲突**提前暴露并达成共识**。

## 必须讨论的议题

### 1. 游戏基调

| 基调类型 | 描述 | 适合的模组 |
|---------|------|-----------|
| 经典恐怖 | 严肃、压抑、角色可能疯狂/死亡 | 死光、永恒恶意 |
| 冒险探索 | 轻松中带紧张，重在解谜 | 墨丘利的庭院 |
| 纯粹欢乐 | 允许出格行为，氛围轻松 | 胖猫巴士 |

### 2. 安全机制

推荐使用以下至少一种：

- **X-Card**：任何人随时举起 X 卡，当前场景立即跳过
- **Lines & Veils**：Lines 是绝对不涉及的内容，Veils 是可以存在但不详细描述的
- **开门政策**：任何人可以不给理由随时离开

### 3. 游戏规则共识

```markdown
需要确认的规则点：
□ PvP 是否允许？
□ 角色死亡后如何处理？（新角色/旁观/退出）
□ 分裂行动的处理方式？
□ 元游戏知识的容忍度？
□ 手机使用规则？
□ 迟到/缺席如何处理？
```

### 4. 角色创建

在 Session Zero 中一起建卡的好处：

1. 确保队伍**技能覆盖**合理
2. 建立角色之间的**关系网络**
3. 守秘人可以**埋入钩子**到角色背景中

## Session Zero 检查清单

- [ ] 确定游戏系统和版本
- [ ] 说明模组类型（单次/短战役/长战役）
- [ ] 讨论并记录安全机制
- [ ] 明确游戏基调
- [ ] 角色创建（或分配创建作业）
- [ ] 确定日程安排（频率、时长）
- [ ] 建立沟通渠道（微信群/Discord/QQ群）

## 常见陷阱

> **不要假设所有人和你想的一样。**
>
> "我以为大家都知道这是恐怖团"
> "我以为角色死了可以直接建新的"
> "我以为我们是认真跑团不是来搞笑的"

每一个"我以为"都是一颗定时炸弹。Session Zero 就是拆弹的时间。

---

*一个好的 Session Zero 可能只需要 30 分钟，但它能拯救未来几十个小时的游戏时间。*
