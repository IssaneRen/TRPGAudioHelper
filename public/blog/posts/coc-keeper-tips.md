# CoC 守秘人进阶技巧

作为一名 CoC（克苏鲁的呼唤）守秘人，掌握一些进阶技巧可以让你的跑团体验更上一层楼。

## 氛围营造

### 环境描写的三层法则

1. **远景** — 先描述整体环境的大氛围
2. **中景** — 聚焦到角色周围的具体事物
3. **近景** — 调动感官细节（气味、温度、声音）

> "你推开那扇锈迹斑斑的铁门，冷风夹杂着咸腥的海水味扑面而来。远处的灯塔在浓雾中若隐若现，像一只独眼巨人窥视着这片荒芜的海岸。"

### 音效使用建议

| 场景 | 推荐音效 | 时机 |
|------|---------|------|
| 调查阶段 | 轻柔的环境白噪音 | 持续播放 |
| 紧张时刻 | 心跳声、低频嗡鸣 | 关键骰点前 |
| 遭遇怪物 | 突然的不协调音 | 描述完毕后 |
| 追逐战 | 快节奏打击乐 | 全程 |

## NPC 扮演

### 快速区分技巧

为每个重要 NPC 准备：

- **一个标志性动作**（比如不停擦眼镜）
- **一个口头禅**（比如"你懂我意思吧"）
- **一个秘密**（即使玩家永远不会发现）

```

```

## 节奏把控

好的守秘人像一个乐队指挥：

1. 开场 **轻松** — 让玩家适应角色
2. 调查 **渐进** — 线索逐步揭示，恐惧缓慢累积
3. 高潮 **爆发** — 真相大白的那一刻
4. 收尾 **余韵** — 给玩家消化的时间

---

*记住：最可怕的不是怪物本身，而是玩家脑海中想象的那个东西。*
