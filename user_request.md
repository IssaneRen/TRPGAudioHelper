# TRPGLuciusHelper - 用户需求文档

## 项目概述

TRPGLuciusHelper 是一个 TRPG（桌面角色扮演游戏）辅助工具项目，以 Web 形式运行，支持本地部署或服务器部署。主要用于支持网络和线下跑团的助手功能。

## 核心需求

### 1. 基础要求

- Web 端提供一个简洁的配置页面，包含多个附属功能
- 需要适配 macOS、Windows、Android、iOS 多平台
- 技术选型偏向前沿技术栈
- 注重文档维护（包括技术文档和 AI 工作记录文档），保证人工随时可以介入

### 2. 核心功能 - 三个 TAB

#### TAB1 - 个人介绍与模组链接

- 展示和维护个人介绍信息
- 管理不同状态的模组：已备过的、待准备的、已经带过的
- 更新带人跑团的免责声明和要求
- 支持随时更新维护

#### TAB2 - 模组工具页面

- 点击具体模组或上传自定义模组，显示可视化的拖拽线索工具
- 支持标注线索之间的关联关系
- 点击线索可记录当前模组中哪些关键线索已被取得（置灰处理）
- 使用有向无环图（DAG）实现线索网络的拖拽与渲染
- 需要较好的图形渲染效果（考虑类似 WebGL 级别的图形技术）

#### TAB3 - 个人博客与杂谈

- 文章更新与展示
- 支持博客和杂谈等多种内容类型

### 3. 数据导入导出

- TAB1 和 TAB2 均支持数据的导入和导出
- 使用 JSON 格式
- 便于在不同设备的 localhost 之间迁移数据
